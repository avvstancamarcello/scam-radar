# Scam Radar

Sito web planetario per il progetto Scam Radar.

## Struttura del Progetto

```
scam-radar/
├── index.html          # Pagina principale
├── css/
│   └── style.css       # Stili CSS
├── js/
│   └── script.js       # Script JavaScript
└── README.md           # Questo file
```

## Come Utilizzare

1. Apri `index.html` nel tuo browser per visualizzare il sito
2. Modifica i file HTML, CSS e JavaScript secondo le tue esigenze
3. Il progetto è pronto per ricevere la tua pagina web personalizzata

## Sviluppo

Il progetto utilizza HTML, CSS e JavaScript vanilla (senza framework) per massima compatibilità e semplicità.

## Note

- Sostituisci il contenuto di `index.html` con la tua pagina web personalizzata
- Aggiungi i tuoi stili in `css/style.css`
- Aggiungi la tua logica JavaScript in `js/script.js`
